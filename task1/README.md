# Task 1 — Responsive Frontend Interface

Personal brand landing page for qvrtik (Kartik Saxena), built as part of the DecodeLabs Full Stack Development Internship 2026.

## What's built
- Responsive landing page with Lando Norris-inspired design
- Full-screen overlay navigation
- Animated CSS blob background (hero section)
- Alternating light/dark sections
- Dev Journal feed (pulls from Task 2 API)
- Portfolio page
- Public log feed (log.html)

## Tech used
HTML5 · CSS3 (Grid + Flexbox) · Vanilla JavaScript · No frameworks

## How to run
Open `index.html` in a browser or use Live Server in VS Code.
Server must be running (Task 2) for the Log feed to populate.
